Deck Quickstart

This is a minimal app that implements Deck using a very basic HTML/vanilla JS frontend with an ASP.NET Core backend.

1. Click "Link Account" and go through the widget. This will submit an access token to your webhook.

2. Confirm in your application logs that an access token was received.
